Calculator Help

Operations

  • + Addition
  • - Subtraction
  • × Multiplication
  • ÷ Division
  • % Modulo (remainder after division)
  • . Decimal point

Keyboard Shortcuts

  • 0-9 Number input
  • + - * / % Operators
  • . Decimal point
  • Enter or = Calculate result
  • Backspace Delete last character
  • Escape All Clear (reset)

Buttons

  • AC All Clear - resets calculator
  • C Clear - clears current input
  • ← Backspace - deletes last character
  • = Calculate - evaluates expression

Special Cases

  • Division by zero shows "Undefined"
  • Modulo by zero shows "Undefined"
  • After calculation, operators continue with result
  • After calculation, numbers start fresh input
